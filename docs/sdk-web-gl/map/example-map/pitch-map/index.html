    Pitch¶

    Pitch được đề cập đến là góc nghiêng của bản đồ so với trục dọc. Nói cách khác, nó xác định cách bản đồ được xoay theo chiều dọc, cho phép bạn thay đổi góc nhìn của bản đồ từ trên xuống theo phương thẳng đứng (góc 0 độ) hoặc từ góc nghiêng (lên đến 60 độ).

    Pitch có thể tạo ra hiệu ứng 3D, giúp nhìn thấy các tòa nhà hoặc địa hình ở một góc nghiêng thay vì nhìn từ trên cao xuống.

    import "./App.css";
import vietmapGl from "map-test-npm";
import "map-test-npm/vietmap-gl.css"
import React, { useEffect, useRef } from "react";
function App() {
  const mapContainer = useRef(null);
  const map = useRef(null);
  const vietmap = { lng: 106.675477, lat: 10.758898 };
  const VIETMAP_API_KEY = "YOUR_VIETMAP_API_KEY_HERE";
  const zoom = 14;
  const pitch = 45;

    useEffect(() => {
    if (map.current) return;

    map.current = new vietmapGl.Map({
        container: mapContainer.current,
        style: "https://maps.vietmap.vn/mt/dm/style.json?apikey=" + VIETMAP_API_KEY, 
        center: [vietmap.lng , vietmap.lat], 
        zoom: zoom,
        pitch: pitch,
      });
       map.current.addControl(new vietmapGl.NavigationControl(), 'top-right');
      }, [vietmap.lng,vietmap.lat,zoom]);

  return (
    <div
      ref = {mapContainer}
      className = "map">
    </div>
  );
}

export default App;

    Prop¶

    Prop Type Default Description Example
    pitch number 0 Góc nghiêng của bản đồ, tính bằng độ. Góc 0 là nhìn thẳng từ trên xuống. pitch: 45
    setPitch() function N/A Thiết lập giá trị pitch (góc nghiêng) của bản đồ. mapRef.current.setPitch(45);
