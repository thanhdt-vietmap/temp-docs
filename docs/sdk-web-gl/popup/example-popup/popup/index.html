    Infowindow¶

    Infowindow is a component that displays information about a specific location on a map.

    Popup Options¶

    Example¶

    1. Install the npm package 
      npm install --save map-test-npm
    2. Import VietMap SDK vào và thêm các hàm ReactJS cần thiết. Chúng ta sẽ thêm những hàm trên vào dòng đầu tiên của App.js. 
      import "./App.css";
import vietmapGl from "map-test-npm";
import "map-test-npm/vietmap-gl.css"
import React, { useEffect, useRef } from "react";
    3. Bây giờ chúng ta sẽ thiết lập trạng thái mặc định cho bản đồ khi khởi tạo.(Example: app.js) 
      function App() {
  const mapContainer = useRef(null);
  const map = useRef(null);
  const vietmap = { lng: 106.675477, lat: 10.758898 };
  const VIETMAP_API_KEY = "YOUR_VIETMAP_API_KEY_HERE";
  const zoom = 14;
}
    4. Lưu trữ các dữ liệu của bản đồ và hiển thị các nút (zoomIn, zoomOut và Location).Hiển thị popup trên bản đồ. 
      useEffect(() => {
  if (map.current) return;

  map.current = new vietmapGl.Map({
    container: mapContainer.current,
    style: "https://maps.vietmap.vn/mt/tm/style.json?apikey=" + VIETMAP_API_KEY,
    center: [vietmap.lng, vietmap.lat],
    zoom: zoom,

    .addTo(map.current);

}, [vietmap.lng, vietmap.lat, zoom]);

    6. Tổng hợp tất cả các phần thành một ví dụ hoàn chỉnh. 
      import './App.css';
import React, { useEffect, useRef } from "react";
import * as vietmapGl from 'map-test-npm';
import "map-test-npm/vietmap-gl.css"

function App() {
  const mapContainer = useRef(null);
  const map = useRef(null);
  const vietmap = { lng: 106.675477, lat: 10.758898 };
  const VIETMAP_API_KEY = "YOUR_VIETMAP_API_KEY_HERE";
  const zoom = 14;

  useEffect(() => {
    if (map.current) return;

    map.current = new vietmapGl.Map({
      container: mapContainer.current,
      style: "https://maps.vietmap.vn/mt/tm/style.json?apikey=" + VIETMAP_API_KEY,
      center: [vietmap.lng, vietmap.lat],
      zoom: zoom,
    });
    map.current.addControl(new vietmapGl.NavigationControl(), 'top-right');

    const popup = new vietmapGl.Popup({ closeOnClick: false })
    .setLngLat([106.67545475079355, 10.758898])
    .setHTML('<h4>VIETMAP JSC - Số 3 đường Trần Nhân Tôn, phường 9, quận 5, Thành Phố Hồ Chí Minh.</h4>')
    .addTo(map.current);

  }, [vietmap.lng, vietmap.lat, zoom]);

  return (
    <div
      ref={mapContainer}
      className="map">
    </div>

  );

}

export default App;
    7. Bây giờ chúng ta sẽ cần tạo kiểu đơn giản để hiển thị bản đồ. Thêm thành phần vào css để điều chỉnh style cho bản đồ. 
      *{
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

*html, body{
  width: 100%;
  height: 100%;

}
.map {
  position: absolute;
  width: 100%;
  height: 100%;
}

    Parameters¶

    Prop Type Default Required Description
    closeButton Boolean true false Xác định có hiển thị nút đóng trên popup hay không.
    closeOnClick Boolean true false Xác định có đóng popup khi nhấp vào bản đồ hay không.
    anchor String auto false Định vị trí của popup so với điểm `.
    offset Number / Object 0 false Khoảng cách giữa popup và điểm neo.
    className String null false Thêm lớp CSS tùy chỉnh vào popup.
    maxWidth String 240px false Đặt chiều rộng tối đa của popup.

    Explain¶

    closeButton¶

    Là nút button đóng của popup nếu như là true thì sẽ hiển thị nút đóng còn flase thì sẽ không hiển thị. Mặc định của closeButton trong popup là true. 

     const popup = new vietmapGl.Popup({ closeButton: false })
    .setLngLat([106.67545475079355, 10.758898])
    .setHTML('<h4>VIETMAP JSC - Số 3 đường Trần Nhân Tôn, phường 9, quận 5, Thành Phố Hồ Chí Minh.</h4>')
    .addTo(map.current);

    closeOnClick¶

    Xác định có đóng popup khi nhấp vào bản đồ hay không. Nếu là true thì khi bấm bất kỳ ở đâu trên bản đồ nó sẽ đóng popup lại. 

     const popup = new vietmapGl.Popup({ closeOnClick: true })
    .setLngLat([106.67545475079355, 10.758898])
    .setHTML('<h4>VIETMAP JSC - Số 3 đường Trần Nhân Tôn, phường 9, quận 5, Thành Phố Hồ Chí Minh.</h4>')
    .addTo(map.current);

    anchor¶

    Ví trí của popup so với điểm neo.

    Value Description
    center Giữa điểm neo
    top Trên điểm neo
    bottom Dưới điểm neo
    left Bên trái điểm neo
    right Bên phải điểm neo
    top-left Góc trên bên trái của điểm neo
    top-right Góc trên bên phải của điểm neo
    bottom-left Góc dưới bên trái của điểm neo
    bottom-right Góc dưới bên phải của điểm neo 
     const popup = new vietmapGl.Popup({ anchor: 'bottom-left' })
    .setLngLat([106.67545475079355, 10.758898])
    .setHTML('<h4>VIETMAP JSC - Số 3 đường Trần Nhân Tôn, phường 9, quận 5, Thành Phố Hồ Chí Minh.</h4>')
    .addTo(map.current);

    offset¶

    Xác định khoảng cách giữa popup và điểm neo. Có thể là 1 số được tính bằng pixel hoặc 1 đối tượng {x:y} để tùy chỉnh cho từng hướng 

    const popup = new vietmapGl.Popup({ anchor: 'bottom-left', offset: 25 })
    .setLngLat([106.67545475079355, 10.758898])
    .setHTML('<h4>VIETMAP JSC - Số 3 đường Trần Nhân Tôn, phường 9, quận 5, Thành Phố Hồ Chí Minh.</h4>')
    .addTo(map.current);

    className¶

    Thêm lớp css vào popup để điều chỉnh style của popup.

    const popup = new vietmapGl.Popup({ className: 'custom-popup' })
    .setLngLat([106.67545475079355, 10.758898])
    .setHTML('<h4>VIETMAP JSC - Số 3 đường Trần Nhân Tôn, phường 9, quận 5, Thành Phố Hồ Chí Minh.</h4>')
    .addTo(map.current);

    .custom-popup {
  background-color: lightblue;
  border-radius: 8px;
}

    maxWidth¶

    Đặt chiều rộng tối đa cho popup. Nhận các giá trị đơn vị như là px, em, %.

    const popup = new vietmapGl.Popup({ maxWidth: "300px"  })
    .setLngLat([106.67545475079355, 10.758898])
    .setHTML('<h4>VIETMAP JSC - Số 3 đường Trần Nhân Tôn, phường 9, quận 5, Thành Phố Hồ Chí Minh.</h4>')
    .addTo(map.current);
