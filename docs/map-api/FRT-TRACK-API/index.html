    FRT TRACK API

    API testing domain¶

    Refer this link: https://maps.fastmap.vn/trackdev/swagger/index.html

    API detailed description¶

    1. Token: Get access token¶

    • URL: /track/api/auth/token
    • Method:POST
    • Parameter: . username . password
    • Output: 
      {
"access_token": "string", 
"expires_in":86400 //validity period (second), 
"refresh_expires_in": 86400 //validity period (second) , 
"refresh_token": "string", 
"token_type": "Bearer",
"not-before-policy": 0, 
"session_state": "4959d78e-6bc6-41f9-adb3-bc12146e9740", 
"scope": "email profile" 
}

    2. ObjectId: Check whether the driver's ID is in Vietmap's server or not¶

    • URL: track/api/objects/objectId
    • Method: GET
    • Parameter: . objectId: Id of FRT's driver
    • Response 
      //If Id exists
{
  "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "accountId": "d6ae709e-4cd4-4c87-9ef0-0e4e383f2d68",
  "name": "testAPI",
  "token": "123456789",
  "projectId": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "properties": {
    "additionalProp1": "string",
    "additionalProp2": "string",
    "additionalProp3": "string"
  },
  "inactive": false,
  "createdDt": "2023-11-13T07:12:29.693081Z",
  "timestamp": "2023-11-13T07:12:29.693082Z"
}

//If Id does not exist
{
  "type": "https://tools.ietf.org/html/rfc7231#section-6.5.4",
  "title": "Not Found",
  "status": 404,
  "traceId": "0HMV1NAGPLTQE:00000001"
}

    3. Object: Create object(driver)¶

    • URL: /track/api/object/objects
    • Method: POST
    • Body request: 
      { 
"id": "3fa85f64-5717-4562-b3fc-2c963f66a123", //unique id of object
"name": "testAPI", 
"projectId": "3fa85f64-5717-4562-b3fc-2c963f66afa6" //projectId can be null
"token": "123456789", 
"properties": {  //object description (unlimited key value)
"vehicle": "motorbike" 
}
    • Response: 
      { 
"id": "3fa85f64-5717-4562-b3fc-2c963f66a123", 
"accountId": "d6ae709e-4cd4-4c87-9ef0-0e4e383f2d68", 
"name": "testAPI", 
"token": "123456789", 
"projectId": null, 
"properties": { 
"vehicle": "motorbike" 
}, 
"inactive": false, 
"createdDt": "2023-11-11T03:31:47.6336438Z", 
"timestamp": "2023-11-11T03:31:47.6336728Z" 
}

    4. Tracks: Device use this API to post its location to server (Driver App)¶

    • URL: /track/api/track/tracks
    • Method: POST
    • Parameters: . objectId: Device's Id . token: Device's token
    • Body request 
      [ 
{ 
"time": 1699673552000, //unix time (milisecond)
"lng": 106.69742569657183, 
"lat": 10.784564871783019, 
"speed": 40, // object speed
"heading": 30, // the direction in which an object is moving (degree), with 0 degerees representing true north and the heading value indicates the angle which the object is moving relative to north
"metaData": {  //object description (unlimited key value)
"Giao hang": "MD01" 
} 
} 
]

    5. Last_wayspoints: API to retrieve the last known information of object sent to server (Customer app/web link)¶

    • URL: /track/api/object/last-waypoints
    • Method: POST
    • Parameter . time: specific time for object's location retrieval
    • Body request: 
      [ 
"3fa85f64-5717-4562-b3fc-2c963f66a123" //object id (driver id)
]
    • Response: 
      { 
"3fa85f64-5717-4562-b3fc-2c963f66a123": { 
"time": 1699673552000 // unix time (milisecond)
"lng": 106.69742569657183 
"lat": 10.784564871783019 
"speed": 40 //speed (km/h)
"heading": 30 // the direction in which an object is moving (degree), with 0 degerees representing true north and the heading value indicates the angle which the object is moving relative to north
"metaData": { 
"Giao hang": "MD01" 
} 
} 
}
