    Marker¶

    Marker là dùng để hiển thị các điểm cụ thể trên bản đồ, thường là để đánh dấu vị trí quan trọng như điểm đến, điểm bắt đầu hoặc các vị trí có sự kiện đặt biệt. Marker có thể tùy chỉnh và dễ dàng thêm vào bản đồ với hình ảnh hoặc các icon.

    Create Marker¶

    Đây là cách để bạn khởi tạo ra 1 marker ở 1 vị trí cố định. Làm theo hướng dẫn này bạn sẽ tạo ra 1 marker được đánh dấu bằng vị trí bạn mà muốn, để hiển thị vị trí đó trên bản đồ.

    Add Marker (Normal)¶

    import './App.css';
import vietmapgl from "./vietmap-gl";
import React, { useEffect, useRef } from "react";
function App() {
  const mapContainer = useRef(null);
  const VIETMAP_API_KEY = "YOUR_VIETMAP_API_KEY_HERE";

  useEffect(() => {
    const map = new vietmapgl.Map({
      container: mapContainer.current,
      style: "https://maps.vietmap.vn/mt/tm/style.json?apikey=" + VIETMAP_API_KEY,
      center: [106.67547751224498, 10.758898464351262],
      zoom: 12,
    });
    map.addControl(new vietmapgl.NavigationControl(), 'bottom-right');

    const marker = new vietmapgl.Marker()
      .setLngLat([106.67547751224498, 10.758898464351262])
      .addTo(map);
    }, []);

  return (
      <div
        ref={mapContainer}
        id="map"
        style={{ width: "100%", height: "100%", position: "relative" }}
      />
  );
}

export default App;

    Add Marker (URL)¶

    Đây cách thêm hình ảnh PNG vào bản đồ từ URL bên ngoài và hình ảnh đó được hiển thị trên bản đồ. 

    import './App.css';
import vietmapgl from './vietmap-gl';
import React, {useEffect, useRef} from 'react';

function App() {
  const mapContainer = useRef(null);

  const VIETMAP_API_KEY = "YOUR_VIETMAP_API_KEY_HERE";

  useEffect(() => {
    const map = new vietmapgl.Map({
      container: mapContainer.current,
      style: "https://maps.vietmap.vn/mt/tm/style.json?apikey=" + VIETMAP_API_KEY,
      center: [106.675477, 10.758898],
      zoom: 12,
    });
    map.addControl(new vietmapgl.NavigationControl(), 'bottom-right');

    map.on('load', async () => {
      map.loadImage('https://bizweb.dktcdn.net/100/415/690/files/thiet-bi-dan-duong-11-jpeg.jpg?v=1676444305147', (error, image) => {

        map.addImage('image-vietmap', image);
        map.addSource('point', {
          'type': 'geojson',
          'data': {
            'type': 'FeatureCollection',
            'features': [
              {
                'type': 'Feature',
                'geometry': {
                  'type': 'Point',
                  'coordinates': [106.675477, 10.758898] 
                }
              }
            ]
          }
        });

        map.addLayer({
          'id': 'points',
          'type': 'symbol',
          'source': 'point',
          'layout': {
            'icon-image': 'image-vietmap',
            'icon-size': 0.25
          }
        });
      });
    });
  }, []);

  return (
    <div
      ref={mapContainer}
      id="map"
      style={{ width: "100%", height: "100%", position: "relative" }}
    ></div>

  );
}

export default App;

    Marker Options¶

    Parameters¶

    Prop Type Default Description
    element HTMLElement undefined HTML element tùy chỉnh để sử dụng làm marker thay vì marker mặc định.
    anchor string "center" Xác định điểm neo của marker.
    lngLat [number, number] undefined Tọa độ địa lý để xác định vị trí marker trên bản đồ, định dạng là [longitude, latitude].
    offset [number],[number] [0, 0] Khoảng cách giữa vị trí marker và tọa độ của nó. Có thể là một số hoặc mảng [x, y].
    draggable boolean false Marker có thể kéo được hay không.
    clickTolerance number false Khoảng cách tối đa tính bằng pixel mà người dùng có thể di chuyển chuột trong khi giữ nguyên lần nhấp.
    rotation number 0 Xoay marker theo góc độ (độ).
    rotationAlignment string "auto" Xác định cách marker xoay. Giá trị có thể là "map", "viewport" hoặc "auto".
    pitchAlignment string "auto" Cách marker điều chỉnh theo độ nghiêng của bản đồ. Giá trị có thể là "map", "viewport"hoặc "auto".
    scale number 1 Tỷ lệ của marker, thay đổi kích thước của nó.
    popup Popup undefined Popup đính kèm với marker, được hiển thị khi người dùng tương tác với marker.
    color string "#3FB1CE" Màu sắc của marker nếu sử dụng marker mặc định, dùng mã màu CSS hợp lệ.

    Explain¶

    Element¶

    Thuộc tính Element cho phép sử dụng một phần tử HTML tùy chỉnh làm marker thay vì sử dụng mặc định. Trong ví dụ này, một thẻ div có hình ảnh marker được tạo và thêm vào bản đồ. 

    useEffect(() => {
    const markerElement = document.createElement('div');
    markerElement.className = 'custom-marker';
    markerElement.style.backgroundImage = 'url(https://example.com/marker.png)';
    markerElement.style.width = '50px';
    markerElement.style.height = '50px';

    new maplibre.Marker({
      element: markerElement,
    })
      .setLngLat([106.7, 10.76])
      .addTo(map);
  }, [map]);

    Anchor¶

    Là vị trí neo của marker tại tọa độ được chỉ định. Anchor cung cấp nhiều vị trí kiểu neo marker.

    Anchor Value Description
    center Tọa độ nằm chính giữa marker. Đây là giá trị mặc định.
    top Tọa độ nằm ở phần trên cùng của marker.
    bottom Tọa độ nằm ở phần đáy của marker.
    left Tọa độ nằm ở bên trái của marker.
    right Tọa độ nằm ở bên phải của marker.
    top-left Tọa độ nằm ở góc trên bên trái của marker.
    top-right Tọa độ nằm ở góc trên bên phải của marker.
    bottom-left Tọa độ nằm ở góc dưới bên trái của marker.
    bottom-right Tọa độ nằm ở góc dưới bên phải của marker.

    Example 

    const marker = new vietmapgl.Marker({
      anchor: "center",
    })
      .setLngLat([106.67547751224498, 10.758898464351262])
      .addTo(map);

    Offset¶

    Offset giúp thay đổi vị trí hiển thị của marker so với vị trí cố định của thực tế của marker đó. Nó di chuyển toàn bộ marker khỏi điểm gốc của nó dựa trên giá trị x và y được cung cấp (theo đơn vị là pixel).

      x: Di chuyển marker theo trục ngang (trái/phải) - Giá trị dương sẽ di chuyển marker sang phải. - Giá trị âm sẽ di chuyển marker sang trái.
      y: Di chuyển marker theo trục dọc (lên/xuống). - Giá trị dương sẽ di chuyển marker xuống dưới. - Giá trị âm sẽ di chuyển marker lên trên.

    Example 

    const marker = new vietmapgl.Marker({
      offset: [0, -25],
    })
      .setLngLat([106.67547751224498, 10.758898464351262])
      .addTo(map);

    ClickTolerance¶

    Là quy định khoảng cách (tính theo pixel) mà người dùng có thể di chuyển chuột trong khi nhấn mà vẫn coi là một lần nhấp.

    const marker = new vietmapgl.Marker({
    clickTolerance: 5,
    })
      .setLngLat([106.67547751224498, 10.758898464351262])
      .addTo(map);

    Rotation¶

    Thuộc tính rotation giúp xoay marker theo góc độ với vị trí hiển thị trên bản đồ (tính bằng độ). 

    const marker = new vietmapgl.Marker({
    clickTolerance: 5,
    })
      .setLngLat([106.67547751224498, 10.758898464351262])
      .addTo(map);

    RotationAlignment¶

    RotationAlignment là cách marker xoay. CÓ 3 cách xoay trong bản đồ: "map","viewport","auto".

    const marker = new vietmapgl.Marker({
    rotation: 45,
    rotationAlignment: 'map',
    })
      .setLngLat([106.67547751224498, 10.758898464351262])
      .addTo(map);

    PitchAlignment¶

    Là cách marker điều chỉnh theo độ nghiêng của bản đồ.

    • map: marker sẽ nghiêng cùng bản đồ.
    • viewport: marker không nghiêng theo bản đồ mà luôn thẳng đứng trong viewport.
    • auto: chọn tự động. 
      const marker = new vietmapgl.Marker({
    pitchAlignment: 'viewport',
    })
    .setLngLat([106.67547751224498, 10.758898464351262])
    `.addTo(map);

    Scale¶

    Thay đổi kích thước của marker so với marker gốc. Ở đây kích thước sẽ được nhân 2 lần so với gốc.

    const marker = new vietmapgl.Marker({
    scale: 2,
    })
      .setLngLat([106.67547751224498, 10.758898464351262])
      .addTo(map);

    Color¶

    Cho phép bạn chỉnh được màu của marker mặc định. 

    const marker = new vietmapgl.Marker({ 
    color: '#25A18E',
    })
      .setLngLat([106.67547751224498, 10.758898464351262])
      .addTo(map);

    Methods¶

    addTo(map)¶

    Thêm marker vào bản đồ muốn hiển thị. 

    marker.addTo(map);

    getLngLat()¶

    Lấy tọa độ hiện tại của marker. Phương thức này dùng cho khi bạn muốn click 1 điểm trên bản đồ hiển thị marker và tọa độ của vị trí đó. 

    const marker = new vietmapgl.Marker({
      draggable: true,
      color: '#25A18E',
    })
      .setLngLat([106.675477, 10.758898])
      .addTo(map);

    marker.on('dragend', () => {
        const lngLat = marker.getLngLat();
        const longitude = lngLat.lng.toFixed(6);
        const latitude = lngLat.lat.toFixed(6);

        console.log('Kinh độ:', longitude);
        console.log('Vĩ độ:', latitude);
      });

    setLngLat()¶

    Đặt vị trí marker đến vị trí được chỉ định. 

    marker.setLngLat([106.67547751224498, 10.758898464351262])

    remove()¶

    Loại bỏ marker ra khỏi bản đồ. 

    marker.remove();

    setElement¶

    Thay đổi phần tử DOM được sử dụng làm marker. Cho phép tùy chỉnh giao diện của marker bằng cách sử dụng các phần tử của HTML. 

        const markerElementEnd = document.createElement('div');
    markerElementEnd.className = 'custom-marker-end';

    const newMarkerEnd = new vietmapgl.Marker({
        element: markerElementEnd,
        anchor: 'bottom',
    })
        .setLngLat([endCoordinates.lng, endCoordinates.lat])
        .addTo(mapRef.current);

    setMarkerEnd(newMarkerEnd);

    getElement¶

    Lấy phần tử DOM hiện tại được sử dụng làm marker.

    HTMLElement: element

    setDraggable(draggable: boolean): this¶

    Cho phép kéo và thả marker trên bản đồ. Khi mà draggable bằng true thì có thể kéo và thả marker trên bản đồ.

    marker.setDraggable(true);

    isDraggable()¶

    Kiểm tra xem marker có đang ở chế độ kéo thả hay không. Trả về true nếu có thể kéo được điểm đánh dấu. 

    const draggable = marker.isDraggable();
console.log(draggable);

    on(event: string, handler: Function): this¶

    Gán một xử lý sự kiện cho marker. Điều này cho phép thực hiện các hành động khi tương tác với marker.

    Parameter - event: Tên sự kiện - handler: Hàm sẽ gọi sự kiện xảy ra.

    marker.on('dragend', () => {
    const lngLat = marker.getLngLat();
        console.log('Kinh độ:', longitude);
        console.log('Vĩ độ:', latitude);
      });

    off(event: string, handler: Function): this¶

    Hủy bỏ sự kiện đã được gán cho marker. Parameter - event: Tên sự kiện - handler: Hàm sẽ gọi sự kiện xảy ra.

    const handleClick = () => {
  console.log('Click Marker');
};
marker.on('click', handleClick);

marker.off('click', handleClick);

    setRotation(rotation: number): this¶

    Cho phép đặt độ xoay marker khi hiển thị trên bản đồ. 

    marker.setRotation(45); // Được tính bằng độ

    getRotation(): number¶

    Lấy ra độ xoay của marker hiện tại đang hiển thị trên bản đồ. 

    const rotation = marker.getRotation();
console.log(rotation);
